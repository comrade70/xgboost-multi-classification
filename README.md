## Project Overview

The Spotify dataset contains various attributes describing songs available on the Spotify platform, including information such as the `artist name`, `album title`, and `track name`. In addition to these descriptive fields, the dataset includes several audio features (numeric columns) that capture musical characteristics, such as `tempo`, `energy`, `danceability`, and `loudness`.

The objective of this project is to demonstrate how audio features (numeric columns) can be used to classify songs into track genres, showcasing a practical application of machine learning in music. The machine learning technique used is the **XGBoost algorithm**

The workflow includes:

- Reading data
- Data cleaning
- Feature selection
- Label encoding of genres
- Train-test data splitting
- Model training using **XGBoost**
- Hyperparameter tuning
- Model evaluation
- Feature importance visualisation
